# Add Bani Captions to vMix

Show Bani Captions on Your Live Stream With Shabad OS + vMix

<CopyOverlayUrl />

## Find the Output Size in vMix

<ResolutionExplanation />

1. Open **Settings**, then click on **Display**
1. Note down the **Output Size**
1. Close the window.

## Add a web browser to the input bar

<Media src='/media/add-browser-vmix.png' alt='vMix' />

1. In the Main Window, use the **Add Input** menu, then click **More**.
1. Click on **Web Browser** in the left pane.
1. Use the Overlay URL from Shabad OS to fill in the **URL**
1. Use the dimensions of the Output Size to fill the **Width** and **Height**.

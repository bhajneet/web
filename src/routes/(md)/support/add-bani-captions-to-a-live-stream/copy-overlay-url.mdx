## Copy the Overlay URL for Shabad OS

You can use the Overlay URL to display gurbani captions.

<Media
  src='/media/settings-overlay-url.png'
  alt='Image of where to find the overlay URL in shabad OS'
>
  Shabad OS > Settings > Overlay > Overlay URL
</Media>

1. Open the **Shabad OS** app on a computer
1. Use the cog/gear icon in the Controller to open **Settings**
1. Go to **Overlay**, then click the **Overlay URL**

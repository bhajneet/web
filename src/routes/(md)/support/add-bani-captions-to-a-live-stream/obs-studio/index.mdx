# Add Bani Captions to OBS Studio

Show Bani Captions on Your Live Stream With Shabad OS + OBS Studio

<CopyOverlayUrl />

## Find the Base (Canvas) Resolution in OBS Studio

<ResolutionExplanation />

1. In OBS Studio, choose **File > Settings**.
1. In the Settings window, click **Video** in the left pane.
1. Note the **Base (Canvas) Resolution**.
1. Close the Settings window.

## Add a browser to the active scene in OBS Studio

You can use a Browser to view the Overlay URL from Shabad OS.

<Media src='/media/add-browser-obs-studio.png' alt='OBS Studio' />

1. In **Sources**, click the **+** button, then add a **Browser** to the scene.
1. In the Browser window, use the Overlay URL from Shabad OS to fill in the **URL**.
1. Use the dimensions of the Base (Canvas) Resolution to fill in the **Width** and **Height**.
1. Delete any text in **Custom CSS**.
1. Close the Browser window.
